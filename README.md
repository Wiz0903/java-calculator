# Java Menu Calculator

A simple interactive calculator built in Java that supports addition, subtraction, multiplication, and division through a text-based menu.

## Features
- Start with an initial number
- Choose operations from a clear menu:
  - Add
  - Subtract
  - Multiply
  - Divide
  - Exit
- Displays final result after exiting

## How to Run
1. Make sure you have **Java JDK** installed
2. Compile the program:
   ```bash
   javac Calculator.java
